# Greece Golden Visa Calculator

A comprehensive calculator for estimating the total investment costs for Greek residency by investment (Golden Visa).

## Quick Start

### Option 1: Deploy to Vercel (Recommended)

1. Push this folder to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project"
4. Select your GitHub repository
5. Click "Deploy"

Done! Your calculator will be live in ~2 minutes.

### Option 2: Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Option 3: Deploy to Netlify

## Features

- Complete cost breakdown for Greece Golden Visa
- All property acquisition costs (transfer tax, notary, lawyer, registration)
- Residence permit costs (application, cards, health insurance, translation)
- Optional services (express processing, power of attorney)
- Interactive cost visualization
- Responsive design
- Law 5100/2024 compliant
